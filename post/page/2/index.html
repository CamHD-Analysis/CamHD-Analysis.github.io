It’s now a sub-directory of https://github.com/amarburg/go-lazycache-app. I’ve done an imperfect modification of the text below… For totally arbitrary, er, I mean perfectly reasonable reasons, I’ve decided to target Google Cloud Platform as my strawman “commercial cloud” provider. I still believe there’s a use case for deploying the software — or some subset of it — on a local or lab machine, but it certainly makes a lot of sense to have the server capable of running in the cloud, whether it’s public or strictly private to service a set of local compute instances, so this seemed like a good place to start.
